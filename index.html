개인정보처리방침

시행일: 2026년 3월 16일  |  최종 수정일: 2026년 3월 16일

핵심 요약

• 밥풀 노트는 인터넷을 사용하지 않는 완전 로컬 앱입니다.

• 수집된 모든 정보는 오직 사용자 기기 내부에만 저장됩니다.

• 개인정보를 외부 서버, 제3자, 광고 네트워크에 절대 전송하지 않습니다.

• 접근성 서비스는 화면 전환 감지 목적으로만 사용하며, 입력 내용·비밀번호·개인 콘텐츠를 읽거나 저장하지 않습니다.

제1조 개인정보처리방침의 목적

밥풀 노트(이하 "앱")는 사용자의 개인정보를 소중히 여깁니다. 본 방침은 앱이 수집·이용하는 정보의 범위, 목적, 보관 방법을 투명하게 안내하기 위해 작성되었습니다. 앱 개발자(이하 "운영자")는 「개인정보 보호법」 및 관련 법령을 준수합니다.

제2조 수집하는 정보 항목 및 목적

앱은 다음과 같이 구분하여 정보를 처리합니다.

(1) 사용자가 직접 입력하는 정보 — 기기 내 저장

항목 내용 목적
노트 내용 사용자가 입력한 텍스트 노트 표시 및 편집
노트 배경색 선택한 색상값(RGB) 노트 외관 유지
노트 유형 항상 표시 / 홈화면 / 특정 화면 노트 표시 조건 관리
노트 위치 화면 내 x·y 좌표, 크기 마지막 위치 기억
노트 상태 최소화 여부, 위치 고정 여부, 표시 여부 노트 상태 복원
생성·수정 일시 타임스탬프 노트 관리

위 정보는 앱 내부 데이터베이스(SQLite/Room, 파일명: babpool_note.db)에만 저장되며, 외부로 전송되지 않습니다.

(2) 접근성 서비스를 통해 일시적으로 처리되는 정보

항목 내용 목적 저장 여부
포그라운드 앱 패키지명 현재 실행 중인 앱의 패키지명
(예: com.example.app)		 해당 앱에 귀속된 노트 표시 노트에 귀속 시 DB 저장
그 외 일시 처리 후 즉시 폐기
포그라운드 액티비티명 현재 화면의 클래스명 화면별 노트 구분 표시 위와 동일
홈화면 페이지 번호 현재 홈화면 페이지 (숫자) 페이지별 노트 표시 미저장, 일시 처리 후 폐기

접근성 서비스가 읽지 않는 정보:

• 키보드 입력 내용, 비밀번호, 사용자가 입력하는 텍스트

• 화면에 표시된 개인 콘텐츠(메시지, 사진, 결제 정보 등)

• 앱 사용 이력, 클릭 위치, 스크롤 내용

• 다른 앱의 데이터베이스나 파일

제3조 개인정보의 보유 및 이용 기간

앱이 처리하는 모든 정보는 사용자가 직접 삭제하거나 앱을 삭제할 때까지 기기 내에 보관됩니다. 운영자는 별도의 서버에 정보를 보관하지 않습니다.

  • 노트 데이터: 사용자가 삭제하거나 앱 삭제 시 함께 제거
  • 접근성 이벤트 데이터: 처리 즉시 폐기 (지속 저장 없음)

제4조 개인정보의 제3자 제공

운영자는 사용자의 개인정보를 어떠한 제3자에게도 제공하지 않습니다. 앱은 인터넷 권한(INTERNET)을 선언하지 않으며, 외부 서버로의 데이터 전송 기능이 전혀 없습니다.

제5조 앱이 사용하는 권한 상세

권한 1 다른 앱 위에 표시 (SYSTEM_ALERT_WINDOW)

다른 앱이 실행 중일 때 노트를 화면 위에 플로팅 창으로 표시하기 위해 필요합니다. 이 권한은 노트 표시 외의 목적으로 사용되지 않습니다.

권한 2 접근성 서비스 (BIND_ACCESSIBILITY_SERVICE)

현재 사용 중인 앱과 화면을 감지하여 해당 화면에 귀속된 노트를 자동으로 표시하기 위해 사용합니다. 접근성 서비스는 Android 시스템이 별도로 관리하며, 사용자가 시스템 설정에서 직접 활성화·비활성화할 수 있습니다. 앱은 이 서비스를 통해 화면 전환 이벤트(TYPE_WINDOW_STATE_CHANGED)와 홈화면 스크롤 이벤트(TYPE_VIEW_SCROLLED)만 감지합니다.

권한 3 진동 (VIBRATE)

플로팅 노트를 길게 누를 때 햅틱 피드백(진동)을 제공합니다. 진동 사용 외의 데이터가 수집되지 않습니다.

권한 4 알림 (POST_NOTIFICATIONS)

접근성 서비스가 활성화된 동안 상태 표시줄에 서비스 실행 알림을 표시합니다. Android 정책에 따라 포그라운드 서비스는 알림을 표시해야 합니다. 알림을 통해 사용자 데이터가 수집되지 않습니다.

제6조 클립보드 사용

앱은 사용자가 노트 내용을 직접 복사할 때 Android 클립보드를 사용합니다. 앱은 클립보드 데이터를 자동으로 읽거나 저장하지 않으며, 사용자의 명시적 동작(복사 버튼 탭)이 있을 때만 노트 내용을 클립보드에 쓰는 동작만 수행합니다.

제7조 광고 및 분석 도구 미사용

앱은 광고 SDK, 분석 도구(Analytics), 충돌 리포트 도구(Crashlytics 등), SNS 연동 SDK를 일절 사용하지 않습니다. 앱에 포함된 외부 라이브러리는 아래와 같으며, 모두 개인정보를 처리하지 않는 로컬 기능 라이브러리입니다.

  • Room (Android Jetpack): 로컬 데이터베이스
  • Hilt (Google): 의존성 주입 (실행 시 메모리 관리)
  • Kotlin Coroutines: 비동기 처리
  • Material Components (Google): UI 컴포넌트

제8조 아동 개인정보 보호

본 앱은 만 14세 미만 아동을 대상으로 개인정보를 수집·이용하지 않습니다. 앱은 연령 확인 기능이 없으며, 만 14세 미만의 아동은 보호자의 동의 하에 앱을 사용해야 합니다.

제9조 개인정보의 안전성 확보 조치

  • 앱 데이터는 Android 앱 샌드박스 내에 저장되어 다른 앱이 접근할 수 없습니다.
  • 운영자는 데이터를 외부 서버에 저장하지 않으므로 서버 해킹에 의한 정보 유출 위험이 없습니다.
  • 기기 분실·도난 등으로 인한 위험은 사용자의 기기 잠금 설정으로 보호받을 수 있습니다.

제10조 개인정보처리방침 변경

본 방침은 관련 법령 변경 또는 앱 기능 추가·변경 시 업데이트될 수 있습니다. 변경 사항은 앱 내 공지 또는 앱 업데이트를 통해 안내하며, 중요한 변경이 있을 경우 앱 첫 실행 시 동의를 다시 요청합니다.
